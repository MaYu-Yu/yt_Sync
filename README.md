# 🛠️ Mayuyu's Gadget - 個人整合工具箱

一個基於 **Flask** 開發的輕量級個人工具整合平台。採用現代化、潮流感的 UI 設計，旨在將日常瑣碎的任務自動化。目前已整合強大的 YouTube 同步管理器，未來將持續擴充更多實用工具。

---

## 🚀 核心工具：YouTube 同步管理器 (Sync Manager)

專為影音收藏與音樂愛好者設計。只需輸入頻道 ID，即可自動追蹤、解析並將影片備份至本地空間。

### 🌟 功能亮點
* **⚡ 免安裝 FFmpeg**：程式啟動時會自動偵測環境，若缺少 FFmpeg，將自動從 GitHub 下載最新 master 分支版本並完成本地配置，無需手動設定系統環境變數。
* **🔍 頻道與清單自動解析**：輸入一個頻道 ID，自動抓取該頻道下所有公開的播放清單，方便快速管理。
* **🔄 增量備份 (Incremental Sync)**：內建 `download_archive` 歷史記錄技術，僅下載新增影片，自動跳過已存在的檔案，節省硬碟空間。
* **🎵 多格式支援**：可自由選擇「最高畫質 MP4」或「高品質 MP3 (192kbps)」轉檔模式。
* **📊 即時狀態監控**：前端進度條顯示當前下載進度、已完成數量與即時系統訊息，下載狀態一目了然。

## 🛠️ 安裝與啟動

### 1. 環境需求
* **作業系統**：Windows 10 / 11 (為了支援內建 `tar` 解壓指令)。
* **Python 版本**：3.8 或以上。

### 2. 安裝套件
請確保你已在專案根目錄，並執行以下指令安裝必要依賴：
```bash
pip install -r requirements.txt
```
### 3. 啟動服務
執行主程式以啟動 Flask Server：
```bash
python app.py
```
啟動後，請於瀏覽器訪問：http://127.0.0.1:5000
